Details - Changed file upload

Changed the server-side file upload situation, so now my developers can upload new server files through the github repository and do not have to ssh into my server. This change has made the process more efficient and secure, eliminating the need for developers to directly access the server.

Benefits

This change has several benefits:

  • Efficiency: Developers can now focus more on coding rather than dealing with server access issues.
  • Security: By limiting direct server access, the security of the server is enhanced.
  • Scalability: This change makes it easier to manage larger teams of developers, as they can all upload files independently.
Future plans

We plan to continue improving our file upload process, with the goal of making it even more seamless and efficient for our developers. Stay tuned for more updates!
